# MiniPGP Test Files

These files are small, human-readable messages for manual GUI testing.

| File | Description |
| --- | --- |
| `01_short_serbian.txt` | Short Serbian message. |
| `02_short_english.txt` | Short English message. |
| `03_long_serbian.txt` | Longer Serbian message. |
| `04_long_english.txt` | Longer English message. |

After receiving a PGP file, compare the displayed or saved plaintext with the
original input file.
